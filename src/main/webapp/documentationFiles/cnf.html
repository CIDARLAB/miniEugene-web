Disjunction of Constraints

miniEugene supports the conjunctive normal form, i.e., a conjunction of disjunctions. A line feed indicates a conjunction. In one line, it is possible to specify disjunctive constraints.

Examples:
1. The desired design must contain a, and b, and c or d.

CONTAINS a
CONTAINS b
CONTAINS c OR CONTAINS d
2. All components in the design must be either forward or reverse oriented.
ALL_FORWARD OR ALL_REVERSE
