miniEugene Downloads

Dear miniEugene users,

we provide miniEugene as
  • a Web Application allowing you to experience and learn the provided rules.
  • a JAR file enabling to embed miniEugene into your Java application.
    To facilitate its usage, we provide a Javadoc and Java examples in our miniEugene's github repository.
  • an XML-RPC Web service enabling to invoke miniEugene over the Internet.
    To facilitate its usage, we provide a Javadoc. Examples how to invoke the miniEugene XML-RPC (using the Apache XML-RPC Library) can be found here.
miniEugene is open source, free to use, extend, improve, and distribute.
Please contact us for further information, questions, or suggestions!

Enjoy miniEugene!
